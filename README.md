lefoy-sublime-theme
===================

My Sublime Text 3 theme

## Installation

Unzip the theme inside your `Packages` folder and add this inside your configuration file:

    "color_scheme": "Packages/Theme - lefoy/lefoy.tmTheme",
    "theme": "lefoy.sublime-theme"
